## Overview

Deep Researcher is my personal project that I've built to learn langgraph technology and how AI agents are actually built.

[Telegram Bot](https://t.me/lusp_research_bot)

---

## Main Features

- We're making our data analyzis by creating AI "roleplay" - first agent asks questions and second response based on sources that we found in the internet.

- Our bot citates every paragraph, so there is almost no galucinations and imagination of the model itself.

- I created telegram wrapper, so anyone can test the bot easily.

- Responses in .md files

- We combine both Hack Club AI and OpenAI API's.
